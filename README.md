# Drone Mission Planner 🛸

An automated flight route planning tool designed for precision agriculture workflows. It calculates optimized coverage tracks and verifies routes against national aviation parameters.

## Core Features
* **Boustrophedon Grid Generation:** Automates lawnmower tracking layouts based on sensor sidelap and target altitudes.
* **Aviation Regulation Rules Engine:** Runs programmatic latitude/longitude cross-checks against restricted spaces and ceiling heights.
* **Telemetry Data Exporter:** Saves coordinate pathways directly into universal CSV layouts.
